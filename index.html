PRESENTACIÓN

De manera independiente a las planeaciones didácticas (TPACK, aula invertida, formación on-line, b-learning, principalmente) que incluyan el uso de tecnologías, y que vayan a implementar los estudiantes de las licenciaturas en educación, se podría elaborar una página web, en tal caso es necesario un dominio en la elaboración de éstas. Este dominio requiere conocimientos pedagógicos y técnicos. Este curso cubre una introducción al conocimiento técnico del diseño web. Los conocimientos pedagógicos les indican a los estudiantes que deben iniciar con objetivos educativos claros, contenidos y actividades que les ayuden a conseguirlos. Esto les corresponde a los participantes, aunque se menciona durante la elaboración técnica.

El diseño web es un tema muy importante visto como un lenguaje bimedia (icónico y tipográfico). El objetivo del diseño web es la elaboración de páginas que cubran principalmente el equilibrio armónico entre estética, funcionalidad y su contribución al logro de objetivos educativos, los cuales son aspectos que condicionan el diseño web.

Técnicamente existen muchas maneras de elaborar una página web. Las páginas web actuales requieren ser: adaptables a las pantallas de todos los dispositivos que las visitan, de fácil navegación, conectadas a las redes sociales, con íconos y con una tipografía de fácil lectura, pero principalmente apoyar al docente para conseguir los objetivos educativos planeados.

En este curso no se toman en cuenta las plataformas educativas Moodle, Edmodo ni canvas porque no dan oportunidad de crear un diseño web. Las ventajas del diseño web permiten mayor libertad para personalizar las presentaciones web.


COMPETENCIAS PREVIAS.

• Habilidad para planear la presentación de un contenido educativo: adecuado al nivel de estudios, ordenado de menor a mayor complejidad, con imágenes adecuadas, principalmente.  


• Capacidad de trabajo en equipo.  


• Capacidad de conceptualizar y formalizar objetivos educativos para traducirlos a imágenes, texto, videos y otras maneras de entender conceptos que se planean en los usuarios de la página web a construir. 

COMPETENCIAS A DESARROLLAR.


• Crear una página web adaptativa, versátil, estética, funcional con contenido educativo. 


• Incluir en la página web una manera de evaluar a sus usuarios. 


• Publicar la página web. 


• Capacidad de aplicar los conocimientos a la práctica. 


• Orientación a resultados. 



COMPETENCIAS INTERPERSONALES. 


• Preparación de trabajos en equipo. 


• Capacidad de crítica y autocrítica. 


COMPETENCIAS ESPECÍFICAS.


• Conocimiento básico del lenguaje HTML: <html>, <a href>, <br>, <img> 


• Conocimiento de las herramientas on-line para crear páginas web: wix, wordpress, mobirise, github, google classroom, google forms, principalmente. 


• Definir la jerarquía de la información mediante la organización de los elementos en la página web. 


• Definir el elemento que se usará en la página web: título, logo, menú, pie de página, imagen, acordeón, carrusel, botones, links, bloques de texto, bloques de imágenes, links a páginas de redes sociales, y links a páginas con contenido PDF, word y power point.  


• Definir links a páginas de google forms para evaluar la participación de los usuarios de las páginas. 

  • PLANEACIÓN DEL CURSO:   ACTIVIDADES A DESARROLLAR.
  • Crear una página simple con un menú. Utilizar el editor de textos notepad y el programa power point.  (Opcional)
  • Crear una cuenta en Github.
  • Seleccionar una materia, tema o tópico de las experiencias educativas que ud. imparte.
  • A partir del tema anterior, crear un mapa conceptual para planear la página web.
  • Buscar imágenes para la página en google. También buscar y subir documentos a la nube "google drive" de archivos  PDF, WORD, POWER POINT del material de su curso.
  • Descargar e instalar la aplicación Mobirise. 
  • Crear o usar una cuenta de google.  
  • Crear la página web en Mobirise.
  • Publicar la página con Github.
  • Crear un formulario en google forms. Incluir preguntas que confirmen la revisión del material publicado en la página web.
  • Reflexión sobre el aprendizaje de los participantes en la sesión sobre el diseño de páginas web.
